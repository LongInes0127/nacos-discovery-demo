# Spring Cloud Alibaba Nacos 服务发现与注册演示

这个项目展示了如何使用 Spring Cloud Alibaba 和 Nacos 实现微服务的服务发现与注册。它提供了一个简单但功能完整的微服务架构示例，非常适合学习和快速上手 Nacos。

## 主要特性:
- 使用 Nacos 作为服务注册中心
- 包含服务提供者和消费者的完整示例
- 演示 Feign 客户端的使用
- 配置文件示例（YAML格式）
- 清晰的项目结构，易于理解和扩展

🚀 完美的 Spring Cloud 微服务入门项目！
